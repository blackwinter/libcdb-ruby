    Revision history for libcdb-ruby¶ ↑

    0.2.0 [2014-12-05]¶ ↑

    • Added encoding support to LibCDB::CDB::Reader.

    • Added LibCDB::CDB#load and LibCDB::CDB#load_file to create a database from a dump.

    • Added LibCDB::CDB#stats and LibCDB::CDB#print_stats to collect stats from a database.

    • Fixed that LibCDB::CDB::Reader#each_key and LibCDB::CDB::Reader#each_value would not return an enumerator when no block was given.

    0.1.1 [2014-04-25]¶ ↑

    • Dropped support for Ruby 1.9.2.

    • Housekeeping.

    0.1.0 [2013-08-03]¶ ↑

    • Removed usage of RARRAY_PTR (cf. Ruby bug #8399).

    • Fixed clobbering errors in CDB.open.

    • Dropped support for Ruby 1.8.7.

    • Housekeeping.

    0.0.4 [2012-08-09]¶ ↑

    • Fixed writing with Ruby 1.9.3 on Windows.

    • Fixed leaking CDB pointer when closing reader.

    0.0.3 [2012-04-02]¶ ↑

    • Fixed linking error on Mac OS X. (Issue #1 reported by minaguib)

    • Some internal refactoring.

    0.0.2 [2012-01-20]¶ ↑

    • Refactored C code to DRY.

    • Added Writer#<< as alias for Writer#store.

    • Added missing delegate CDB#total.

    • A little housekeeping.

    0.0.1 [2012-01-12]¶ ↑

    • Birthday :-)
